# AK_MUSIC_MP3_BOT

🎵 Telegram MP3 Music Bot - Fast + Pro

## Features
- Search gaane by name - 10 result ek saath
- Youtube link se direct MP3 download
- 20+ Mood & Language: `/mood sad` `/mood punjabi` `/mood tamil` `/mood rap`
- Queue system: `/next` `/back` `/stop` `/queue`
- Playlist: `/playlist` - 10 gaane save karo
- Favourite list: `/fav`
- Thumbnail ke saath MP3
- Admin Panel: `/users` `/stats` `/ban` `/unban` `/monthly`
- Reset Proof with Replit DB
- Live user activity alerts

## Commands
`Gaane ka naam` - Search
`YT Link` - Direct Download
`/mood` - Mood list
`/queue` - Show queue
`/playlist` - Show saved playlist
`/fav` - Favourite songs
`/next` `/back` `/stop` - Control

## Admin Commands
`/users` - All user IDs
`/stats` - Total, Monthly, Daily stats
`/monthly` - Monthly report
`/ban ID` - Ban user
`/unban ID` - Unban user
